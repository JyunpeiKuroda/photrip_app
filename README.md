# フォトリップApp
[本番環境](https://photrips.herokuapp.com/login)

# 技術スタック

- **バックエンド**：Laravel 5.8
- **フロントエンド**：Vue.js(SPA), Vue Router, Vuex, tailwindcss, Buefy
- **インフラ**：Heroku

※ [herokuにデプロイした備忘録](https://www.kmmk.work/entry/2019/11/30/030459)

# 作成するアプリケーションの要件

- ユーザーがしおりを作成可能
- しおりは共有可能（タイムライン形式）
- しおりには画像の添付が可能
